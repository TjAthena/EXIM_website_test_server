关于花生的信息

花生，也称为地下坚果，是一种因其营养密度和在各种烹饪应用中的多样性而闻名的豆科作物。它们从技术上讲是豆类，而不是坚果，属于豆科（Fabaceae）家族。花生可以生吃、烤制，或作为花生酱、食用油和酱料的原料。它们在零食中尤其受欢迎，并作为世界许多菜肴中的主食。

鸟用花生

花生供应链：从 农场 到 喂鸟器

种植与收获

在印度，用于鸟类饲料的花生主要种植在古吉拉特邦、拉贾斯坦邦、安得拉邦和泰米尔纳德邦。主要品种包括Runner/Bold和Spanish/Java，受气候、土壤和农业技术的影响。

初步筛选与清洁

花生通过专用机器进行筛选，去除污垢和污染物。只有高质量的花生，如60/70、80/100或切割花生，才会被选中，以避免对鸟类健康的风险，如窒息风险。

转化

花生经过漂白、烘烤或压碎，以提高其风味、营养价值和保质期。烘烤和压碎使其更容易被鸟类，特别是较小的物种消费和消化。

质量控制与分类

花生经过严格的黄曲霉毒素检测，以符合安全标准。质量控制确保花生不含有害污染物，从而保证为鸟类提供安全健康的食品。

包装与分销

花生以不同形式包装，包括散装袋（1至1.25吨）和25公斤真空包装。分销渠道确保产品能够到达零售商，从而满足园艺爱好者和野生鸟类饲养者的需求。


价格比较
鸟类食用花生与人类食用花生

用于鸟类食用的花生通常比人类食用的花生更为实惠，尽管价格是相关的。人类食用花生价格下降时，通常也会导致鸟类食用花生的成本下降，这与供应和产量等因素有关。人类食用的花生有更严格的黄曲霉毒素限量（黄曲霉毒素B1：2 ppb，总量：4 ppb），而鸟类食用的花生需符合欧盟较为宽松的标准，总黄曲霉毒素不得超过20 ppb，这凸显了安全测试的重要性。

JAVA 花生

Java 花生是皮肤浅色且美味的小花生，常用于零食、烘烤和油脂生产，以其温和的味道和多功能性而闻名。

小尺寸、温和的味道、高油含量、浅色皮、多用途。

  • JAVA(西班牙型)
  • JAVA 中型
  • JAVA 长型

TJ 花生

TJ 花生为小至中等大小，以其丰富的风味和高含油量而闻名。它们通常用于烘烤和零食。

小至中等大小、风味丰富、高含油量、多用途、适合烘烤。

  • TJ 大胆型
  • TJ 奔跑型
  • TJ 红皮型

Runner 花生

Runner 花生是中等大小且均匀的花生，以其平衡的风味和多功能性而闻名。广泛用于花生酱和零食，因其一致性受到青睐。

中等大小、形状均匀、风味平衡、高产量、多用途。

  • Runner 特大号
  • Runner 中号

去皮花生

去皮花生是无皮且质地光滑的花生，非常适合烹饪、烘焙和零食。由于味道温和，非常适合用于花生酱和酱料。

无皮、质地光滑、味道温和、多用途、适合烹饪。

  • 大胆型 (弗吉尼亚型)
  • JAVA (西班牙型)
  • Runner (特大号)
  • Runner (中号)
